# ERA5 Legacy Download Backups

This folder holds backup copies of older or transitional ERA5 download
entrypoints from `scripts/downloads/`.

These are copies, not the authoritative active entrypoints. The current
recommended ERA5 download scripts remain in the parent directory and are listed
in:

- `scripts/downloads/README.md`

This backup area exists so the main directory can be simplified later without
losing historical scripts during the transition to the native reduced-Gaussian
ERA5 pipeline.
